## 🚀 COMO USAR

### 1. Carregar Notificações
```dart
Provider.read<NotificacoesProvider>(context).carregarNotificacoes();
```

### 2. Gerar Notificação Manualmente
```dart
await notificacoesProvider.gerarNotificacaoComentario(
  solicitacaoId: '123',
  titulo: 'Novo comentário',
  mensagem: 'Texto...',
  authorId: 'userA',
  authorName: 'João',
  ehInterno: false,
  usuariosParaNotificar: ['userB'],
);
```

### 3. Ver Notificações
```dart
Navigator.pushNamed(context, '/notificacoes');
```

### 4. Listenar Mudanças
```dart
Consumer<NotificacoesProvider>(
  builder: (_, provider, __) {
    return Text('Não lidas: ${provider.totalNaoLidas}');
  },
)
```
