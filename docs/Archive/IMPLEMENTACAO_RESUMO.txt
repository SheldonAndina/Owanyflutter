💡 COMO USAR
═══════════════════════════════════════════════════════════════════════════════

1. Navegar para listagem V2:
   Navigator.pushNamed(context, '/solicitacoes');

2. Acessar dados no provider:
   Consumer<SolicitacoesProviderV2>(
     builder: (context, provider, _) {
       // Usar provider.solicitacoes, provider.isLoading, etc.
     },
   )

3. Filtrar:
   await provider.setFilters(status: 'Pendente');

4. Próxima página:
   await provider.loadNextPage();

5. Criar solicitação:
   await provider.criarSolicitacao(dto);

6. Mudar status:
   await provider.mudarStatus(id, statusDto);

7. Comentar:
   await provider.adicionarComentario(id, comentarioDto);

8. Upload:
   await provider.uploadAnexo(id, fileBytes, fileName);
