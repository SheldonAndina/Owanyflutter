================================================================================
✨ COMPONENTES TOP 5
================================================================================

1. ModernAppBar
   └─ Uso: Todo screen precisa de AppBar
   └─ Antes: AppBar(title: Text('Title'), backgroundColor: Colors.brown)
   └─ Depois: ModernAppBar(title: 'Title')
   └─ Benefício: Rounded corners, status bar integration, menos código

2. ModernButton
   └─ Uso: Ações primárias (orange)
   └─ Antes: ElevatedButton(onPressed: () {}, child: Text('Enviar'))
   └─ Depois: ModernButton(label: 'Enviar', onPressed: () {})
   └─ Benefício: Loading state, ícones, desabilitado, altura padrão

3. ModernCard
   └─ Uso: Container padrão
   └─ Antes: Container(decoration: BoxDecoration(...), child: ...)
   └─ Depois: ModernCard(child: ...)
   └─ Benefício: Elevação, border, padding padrão, tap detection

4. DashboardHeader + MetricCard
   └─ Uso: Estatísticas
   └─ Antes: Vários containers customizados
   └─ Depois: MetricCard(icon: ..., value: ..., label: ...)
   └─ Benefício: Consistência, porcentagem com cor dinâmica

5. ModernBottomNavBar
   └─ Uso: Navegação inferior
   └─ Antes: Nenhum ou BottomNavigationBar padrão
   └─ Depois: ModernBottomNavBar(items: [...], onTap: ...)
   └─ Benefício: Badges, customizável, visual profissional

import 'package:flutter/material.dart';
import 'package:provider/provider.dart';
import '../../theme/owany_theme.dart';
import '../../widgets/modern_components.dart';
import '../../utils/app_logger.dart';

class MyScreen extends StatefulWidget {
  const MyScreen({Key? key}) : super(key: key);
  
  @override
  State<MyScreen> createState() => _MyScreenState();
}

class _MyScreenState extends State<MyScreen> {
  late AppLogger _logger;
  
  @override
  void initState() {
    super.initState();
    _logger = AppLogger();
  }
  
  @override
  Widget build(BuildContext context) {
    return Scaffold(
      appBar: ModernAppBar(title: 'Title'),  ← SEMPRE USAR
      body: Consumer<MyProvider>(
        builder: (context, provider, _) {
          if (provider.isLoading) return _buildLoading();
          if (provider.hasError) return _buildError(provider);
          return _buildContent(provider);
        },
      ),
    );
  }
}

⚠️ NÃO FAÇA (Anti-Patterns)
================================================================================

❌ Usar AppBar(...)
   → Use ModernAppBar(...)

❌ Usar Colors.orange / Color(0xFF...)
   → Use OwanyTheme.primaryOrange

❌ Usar setState(() { })
   → Use Provider.read<Provider>(context).method()

❌ Usar Container com BoxDecoration customizado
   → Use ModernCard(...)

❌ Usar ListView sem ModernListItem
   → Use ModernListItem(...) dentro do ListView

❌ Usar print() para logging
   → Use _logger.info('Tag', 'message')

❌ Duplicar código de componentes
   → Use componentes reutilizáveis
