## 📝 Notas Importantes

### 🔴 API - Endpoints Importantes
- **Base URL**: `https://localhost:7068/api`
- **Endpoints V1**: `/api/Solicitacoes`, `/api/Apartamentos`, `/api/Usuarios`
- **❌ V2 NÃO EXISTE**: `/api/v1/solicitacoesv2` retorna 404
- **Provider Methods**: `loadSolicitacoes()`, `loadSolicitacao()`, `criarSolicitacao()`, `mudarStatus()`

### 🎨 Design - Cores Principais
- **Primary Orange**: `#FF7A3D` (OwanyTheme.primaryOrange)
- **Primary Brown**: `#2D1B0E` (OwanyTheme.primaryBrown)
- **Success**: `#7BA57E` (OwanyTheme.success)
- **Error**: `#E85D46` (OwanyTheme.error)
