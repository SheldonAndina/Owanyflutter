# Owany · Icon Asset Pack

**Brand Color:** `#FF5A30` → `#C82800` (gradient)  
**Style:** 2.5D building — coral/red, dark bg

---

## 📁 Folder Structure

### `/svg` — Vector Masters
| File | Use |
|------|-----|
| `owany-icon.svg` | Master — transparent background |
| `owany-icon-dark-bg.svg` | Dark background rounded square |
| `owany-icon-white-bg.svg` | White background rounded square |

### `/png/transparent` — Transparent PNGs
Sizes: 16, 24, 32, 48, 64, 96, 128, 192, 256, 512, 1024px

### `/png/white-bg` & `/png/dark-bg`
256, 512, 1024px with solid backgrounds

### `/ios` — Apple App Store
All required sizes for iPhone, iPad + App Store 1024px  
*No alpha channel (Apple requirement)*

### `/android` — Google Play
Adaptive icon foreground layers + solid versions  
All mipmap densities: ldpi → xxxhdpi + Play Store 512px

### `/web` — Web & PWA
- `favicon.ico` (16/32/48/64/128/256 multi-size)
- `favicon-16x16.png`, `favicon-32x32.png`
- `android-chrome-192x192.png`, `android-chrome-512x512.png`
- `maskable-512x512.png` (PWA maskable icon)
- `apple-touch-icon-180x180.png`
- `og-image-400x400.png` (Open Graph meta tag)

### `/windows` — Microsoft Store / UWP
SmallTile, Square150, Wide310, LargeTile, StoreLogo, SplashScreen

### `/macos` — macOS App Bundle
Full `.icns` set: 16→1024 @1x and @2x

### `/social` — Social Media
Twitter, LinkedIn, Instagram, Facebook profiles + Slack emoji

### `/pdf` — Print / Figma / Illustrator
Vector PDF — scalable to any size
